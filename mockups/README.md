# Pasta de Mockups

Coloque aqui as imagens dos seus mockups para analisar com o Claude Code.

## Como usar

1. Salve o mockup nesta pasta (PNG, JPG ou WEBP)
2. Abra o Claude Code no terminal
3. Diga: "Analise este mockup: mockups/nome-do-arquivo.png"
4. O Claude vai sugerir copies, publico ideal e formato de anuncio

## Sugestao de organizacao

```
mockups/
  copa-2026/       <- estampas da linha Copa
  brasil/          <- linha Brasil / identidade brasileira
  humor/           <- estampas de humor cotidiano
  torcida/         <- frases de torcida
```

Pode criar subpastas por linha ou deixar tudo aqui — como preferir.
